# Aether

A clean modern mobile web app that turns a concept into a structured creative direction.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   npm start
   ```
3. Open `http://localhost:3000`.

## Notes

- App flow: Input screen -> Loading screen -> Results screen.
- No external APIs are used.
- The app detects concept keywords and returns preset creative directions for:
  - banana
  - cinnamon
  - rocket
  - cotton candy
  - pizza
  - volcano
  - galaxy
- Non-matching concepts return a default modern minimalist direction.
